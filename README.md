# CS419
# Yêu cầu:
  - Python 3.8
  - Nodejs
# Cách sử dụng
  Bước 1: Cài module cần cho FE:
  - cd tới đường dẫn client
  - Gõ npm i trong terminal
  
  Bước 2: Trường ghi
  
  Bước 3: Chạy FE:
  - cd tới đường dẫn client
  - Gõ npm start trong terminal

  Bước 4: Chạy BE:
  - cd tới đường dẫn backend
  - Gõ python main.py
